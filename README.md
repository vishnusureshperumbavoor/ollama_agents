# Ollama Agents

Creating autonomous agents using Ollama, CrewAI and mistral

# Agents
* Blog researcher
* Blog writer

## Prerequisites
* Ollama
* Python

## How to set up
Clone the repo & open folder
```sh
pip install -r requirements.txt
```
Run
```
python app.py
```
